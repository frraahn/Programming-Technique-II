# 🎮 Programming Technique II

Welcome to my repository for the **Programming Technique II** course at UTM! This project displays our team-based game development alongside our practical C++ laboratory exercises.

---

## 📂 Project & Lab Breakdown

### 🔹 Project: Fruit Blitz Game Application
* **Summary:** Our team created an interactive game called **Fruit Blitz**. Before writing the code, we planned the system using flowcharts and class diagrams. The game includes features like score tracking, timers, and game loops to keep players engaged.
* **Files:** 
  * Game Files 
  * Design & Architecture Report

### 🔹 Lab Work: C++ & OOP Exercises
* **Summary:** This is a collection of laboratory exercises focused on learning and applying OOP in C++. The work includes creating basic classes with safe data access[, handling arrays of objects, modifying strings by overloading operators (like `!` and `*`) and linking different classes together using dynamic memory allocation.

---

## 💡 Course Reflection & Key Takeaways

This course helped me understand how to turn a real-world problem into organized code. Making the **Fruit Blitz** game and finishing the lab exercises showed me that planning code before writing it saves a lot of time. 

I learned how to use core coding concepts like **encapsulation** to keep data safe inside objects, and **operator overloading** to make code cleaner. I also practiced tracing memory pointers and managing dynamic arrays without bugs. Overall, I feel much more confident building interactive programs and working on coding projects with a partner!
